1. Régression linéaire simple

Le but d'une régression linéaire est de comprendre comment une variable X (explicative) influence une variable Y (expliquée).

La relation entre ces deux variables est supposée être descriptible par une fonction affine (la droite de régression) :

Y=ωX+bY = \omega X + bY=ωX+b

Ici :

  • X={x1,x2,...,xn}X = \{x_1, x_2, ..., x_n\}X={x1​,x2​,...,xn​} est un vecteur de valeurs de la variable explicative
  • Y={y1,y2,...,yn}Y = \{y_1, y_2, ..., y_n\}Y={y1​,y2​,...,yn​} est un vecteur de valeurs de la variable expliquée
  • ω\omegaω est le coefficient de proportionalité (pente) de la droite de régression
  • bbb est l'ordonnée à l'origine (interception) de la droite de régression

Pour déterminer la droite à utiliser, il existe plusieurs méthodes :

  • Line fitting :

  • Minimisation du coût :

  • Interpolation : prédire des valeurs de YYY pour des valeurs de XXX comprises dans l'intervalle de valeurs de XXX utilisées pour l'apprentissage
  • Extrapolation : prédire des valeurs de YYY pour des valeurs de XXX en dehors de l'intervalle de valeurs de XXX utilisées pour l'apprentissage

2. Evaluation de qualité de modèle

Pour évaluer la qualité d'un modèle de régression, il existe plusieurs métriques qui permettre de mesure la relation entre la variable X et Y.

2.1. Erreur quadratique moyenne (MSE)

L'erreur quadratique moyenne est une mesure caractérisant la précision d'un modèle.

Comme vu précedemment, elle se calcule avec :

MSE=1n∑i=1n(Y^predit(i)−Yreel(i))2MSE = \frac{1}{n} \sum_{i=1}^{n} (\hat{Y}_{predit}(i) - Y_{reel}(i))^2MSE=n1​i=1∑n​(Y^predit​(i)−Yreel​(i))2

2.2. p-value

La p-value est une mesure statistique qui désigne la probabilité d'obtenir un résultat au moins aussi extrême que celui observé, en supposant que l'hypothèse nulle est vraie. En gros, c'est la probabilité que la relation entre les variables X et Y soit due au hasard.
Une valeur faible (inférieure à 0.05) indique que la relation entre les variables est significative.
Une valeur plus importante indique au contraire que nous n'avons pas assez de preuves pour rejeter l'hypothèse nulle (càd pour prouver que ce n'est pas du hasard).

2.3. Corrélation R² et R² ajusté

Le coefficient de détermination R² est une mesure statistique qui indique la proportion de la variance de la variable expliquée qui est expliquée par la variable explicative.

Soit SSTSSTSST la somme totale des carrés des différences entre chaque valeur réelle et la moyenne de Y :

R2=1−SSRSST,R2∈[0,1]R^2 = 1 - \frac{SSR}{SST}, R^2 \in [0,1]R2=1−SSTSSR​,R2∈[0,1]

La valeur de R2R^2R2 peut être interprétée comme suit :

  • R2=1R^2 = 1R2=1 : le modèle explique parfaitement la variance de Y

3. Régression linéaire multiple

3.1 Définition

La régression linéaire multiple est une extension de la régression linéaire simple. Elle permet de modéliser la relation entre une variable expliquée YYY et plusieurs variables explicatives X1,X2,...,XnX_1, X_2, ..., X_nX1​,X2​,...,Xn​.

La relation entre ces variables est décrite par la fonction suivante :

Y=ω1X1+ω2X2+...+ωnXn+bY = \omega_1 X_1 + \omega_2 X_2 + ... + \omega_n X_n + bY=ω1​X1​+ω2​X2​+...+ωn​Xn​+b

Ici, chaque ωi\omega_iωi​ représente le coefficient de la variable explicative XiX_iXi​. Le but est de trouver les valeurs de ωi\omega_iωi​ qui minimisent la somme des carrés des résidus (MSE).

Le calcul des coefficients ωi\omega_iωi​ se fait de la même manière que pour la régression linéaire simple :

ωi=N(∑nXiY)−(∑nXi)(∑nY)N(∑nXi2)−(∑nXi)2=cov(Xi,Y)var(Xi)\omega_i = \frac{N(\sum_n{X_iY})-(\sum_n{X_i})(\sum_n{Y})}{N(\sum_n{X_i^2})-(\sum_n{X_i})^2} = \frac{cov(X_i, Y)}{var(X_i)}ωi​=N(∑n​Xi2​)−(∑n​Xi​)2N(∑n​Xi​Y)−(∑n​Xi​)(∑n​Y)​=var(Xi​)cov(Xi​,Y)​

b=Yˉ−∑i=1mωiXiˉb = \bar{Y} - \sum_{i=1}^{m} \omega_i \bar{X_i}b=Yˉ−∑i=1m​ωi​Xi​ˉ​ (avec mmm le nombre de variables explicatives).

3.2 Calcul matriciel

Il est possible de résoudre le problème de la régression linéaire multiple en utilisant des calculs matriciels.

Soit XXX la matrice des variables explicatives, YYY le vecteur des valeurs de la variable expliquée, ω\omegaω le vecteur des coefficients et bbb l'ordonnée à l'origine. On a alors :

Y=Xω+bY = X\omega + bY=Xω+b

Pour trouver les valeurs de ω\omegaω et bbb qui minimisent la somme des carrés des résidus, on peut utiliser la formule suivante :

avec (XTX)(X^TX)(XTX) la matrice de variance-covariance des variables explicatives

4. Régression logistique

4.1 Définition

L'objectif de la régression logistique est de prédire la probabilité qu'un événement se produise en fonction de variables explicatives. Contrairement à la régression linéaire, la régression logistique est utilisée pour prédire des variables binaire.

On cherche à obtenir un résultat entre 0 et 1, qui représente la probabilité que l'événement se produise. Pour cela, on utilise généralement la fonction sigmoïde :

σ(u)=11+e−u=y^\sigma(u) = \frac{1}{1 + e^{-u}} = \hat{y}σ(u)=1+e−u1​=y^​ avec u=ω1X1+ω2X2+...+ωnXn+bu = \omega_1 X_1 + \omega_2 X_2 + ... + \omega_n X_n + bu=ω1​X1​+ω2​X2​+...+ωn​Xn​+b ici.

4.2 Fonction de coût

Les fonctions de coût les plus utilisées pour la régression logistique sont l'entropie croisée (cross-entropy) et l'erreur quadratique moyenne (MSE).

L'entropie croisée est définie comme :

J(ω)=−1n∑i=1n[yilog(y^i)+(1−yi)log(1−y^i)]J(\omega) = -\frac{1}{n} \sum_{i=1}^{n} [y_i log(\hat{y}_i) + (1 - y_i) log(1 - \hat{y}_i)]J(ω)=−n1​i=1∑n​[yi​log(y^​i​)+(1−yi​)log(1−y^​i​)]

et l'erreur quadratique moyenne comme :

MSE=1n∑i=1n(y^i−yi)2MSE = \frac{1}{n} \sum_{i=1}^{n} (\hat{y}_i - y_i)^2MSE=n1​i=1∑n​(y^​i​−yi​)2

avec yiy_iyi​ la valeur réelle de la variable binaire, y^i\hat{y}_iy^​i​ la valeur prédite par le modèle.

1. Analyse en composantes principales (ACP)

On considère nnn variables X1,X2,...,XnX_1, X_2, ..., X_nX1​,X2​,...,Xn​, chacune de dimension ppp. L'objectif de l'ACP est de réduire la dimension de l'espace des variables en conservant le maximum d'information.

On cherche donc à trouver qqq axes (composantes principales, q<pq<pq<p) qui permettent de représenter les données sans perdre trop d'information.

Etapes de l'ACP :

  1. Centrer les données : Xc=X−XˉX_c = X - \bar{X}Xc​=X−Xˉ,
  2. Calculer la matrice de covariance : V[X]=1nXcTXcV[X] = \frac{1}{n}X_c^TX_cV[X]=n1​XcT​Xc​,
  3. Diagonaliser la matrice de covariance :
    • Calculer les valeurs propres λi\lambda_iλi​ : racines du polynôme caractéristique de V[X]V[X]V[X], det(V[X]−λI)=0det(V[X] - \lambda I) = 0det(V[X]−λI)=0,
    • Déterminer les vecteurs propres viv_ivi​ de V[X]V[X]V[X] : V[X]vi=λiviV[X]v_i = \lambda_i v_iV[X]vi​=λi​vi​,
    • Les vecteurs propres doivent être rangés par ordre décroissant de valeurs propres.
  4. Déterminer la qualité des variables
    • Calculer la quantité totale d'information contenue dans les données : λ=∑i=1pλi\lambda = \sum_{i=1}^{p} \lambda_iλ=∑i=1p​λi​,
    • Calculer la proportion d'information contenue dans chaque axe : qi=λiλq_i=\frac{\lambda_i}{\lambda}qi​=λλi​​,
    • Choisir le nombre d'axes à conserver en fonction de la quantité d'information souhaitée.
  5. Calculer les composantes principales : Y=XcVY = X_cVY=Xc​V, avec V=[v1,v2,...,vk]V = [v_1, v_2, ..., v_k]V=[v1​,v2​,...,vk​] les vecteurs propres retenus.

Les composantes principales calculées sont les axes de l'ACP. Elles permettent de représenter les données dans un espace de dimension réduite.

Le coefficient de corrélation linéaire entre deux variables XiX_iXi​ et XjX_jXj​ est donné par :

rij=cov(Xi,Xj)σij=<Xi,Xj>∥Xi∥∥Xj∥≅cos(Xi,Xj)r_{ij} = \frac{cov(X_i, X_j)}{\sigma_{ij}} = \frac{<X_i, X_j>}{\|X_i\| \|X_j\|} \cong cos(X_i, X_j)rij​=σij​cov(Xi​,Xj​)​=∥Xi​∥∥Xj​∥<Xi​,Xj​>​≅cos(Xi​,Xj​)

On peut alors représenter ces corrélations dans un plan ayant pour axes deux composantes principales. Les variables qui sont proches dans le plan sont fortement corrélées.

3. Analyse factorielle des données mixtes (AFDM)

En gros, c'est quand il y a à la fois des variables quantitatives et qualitatives dans le même dataset. On fait des ACP sur les variables quantitatives et des ACM sur les variables qualitatives.

Exemple :

ID Personne Note Couleur chemise
1 17 R
2 19 V
3 10 R
4 16 B

Comment la couleur est qualitative, on compte une variable différente par possibilité.

Ainsi, on obtient :

ID Personne Note Couleur chemise_R Couleur chemise_V Couleur chemise_B
1 17 1 0 0
2 19 0 1 0
3 10 1 0 0
4 16 0 0 1

Il y a donc désormais 4 variables quantitatives à étudier.
