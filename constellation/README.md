# Constellation Project

This project visualizes 'n' points in a canvas, simulating attraction forces between them to form constellations. 

## Project Structure

```
constellation-project
├── src
│   ├── index.html        # Main HTML document
│   ├── styles.css       # Styles for the constellation display
│   └── scripts
│       └── main.js      # JavaScript code for generating points and animations
└── README.md            # Project documentation
```

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd constellation
   ```

3. **Open the `index.html` file** in your web browser.

## Functionality

- The project generates a specified number of points on a canvas.
- Each point interacts with others through attraction forces, creating dynamic movements.
- The points will gradually form recognizable constellations as they are animated.

## Dependencies

This project does not require any external libraries. It uses standard HTML, CSS, and JavaScript.
